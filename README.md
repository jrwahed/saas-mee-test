# MW Growth Systems

AI-Powered Marketing & Sales Intelligence Dashboard

Built by Mohamed Waheed — [mohamedwaheed.vercel.app](https://mohamedwaheed.vercel.app)

## Stack
- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase (Auth + Database + Realtime)
- Google Gemini AI

## Features
- Real-time Campaign Analytics
- AI Growth Brain (AI Insights Engine)
- CRM with Kanban Board
- Sales Pipeline Tracker
- Automated Reports

## Setup
1. Clone the repo
2. Copy `.env.example` to `.env` and fill in your Supabase + Gemini keys
3. Run `npm install && npm run dev`
